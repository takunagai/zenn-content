ターミナルで動かしている Claude Code 等からコピーしたテキストには、画面幅での折り返しによる改行やインデントが含まれます。
コピペして使う際に邪魔なので、それらを除去して貼り付けたい。そのための方法を紹介します。
なお、段落の区切り（空行）は保持されます。

  独実行したいだけなので、上の「クイックアクションから...」ブロックは不要です。

## 設定

ショートカットアプリで新規ショートカットを作成し、「シェルスクリプトを実行」アクションを追加。
スクリプト欄に以下の3行を入力。
他の設定はデフォルトのままでOK。

```
pbpaste | perl -0777 -pe '
s/\x{3000}/ /g;
s/^[ \t]+|[ \t]+$//gm;
s/(?<!\n)\n(?!\n)/ /g;
s/[ \t]{2,}/ /g;
s/\n{3,}/\n\n/g;
' | pbcopy

sleep 0.1
osascript -e 'tell application "System Events" to keystroke "v" using command down'
```

このショートカットの名前を「Clean Paste」に設定し、右上にある ℹ️ からキーボードショートカット `⌘⌥V`(お好みに)を割り当て。
以上。

## 使い方

テキストをコピーした状態で、エディタに移動し、`⌘⌥V`。これで整形済みテキストが貼り付けされます。

## 初回のアクセス許可

初めて実行する際、初回のみ以下の許可を求められるので許可してください。

- 「"ショートカット"から System Events を制御する許可」
   → 「OK」
- 「○○○.app がアクセシビリティ機能の使用を求めています」
   → 「システム設定を開く」→ 該当アプリのスイッチをオン（なければ +ボタンから追加）→ 再実行
